# Zch UI Of Vue

## 介绍
zch-gulu 是基于 Vue 的 UI 组件库，主要用于研发企业级中后台产品。

## 特性

- 开箱即用的高质量 React 组件。
- 60+ 高质量组件
- 90% 单元测试覆盖率
- 支持按需引入
- 全链路开发和设计工具体系。
- 提炼自企业级中后台产品的交互语言和视觉风格。

## 安装
### npm 安装
推荐使用 npm 的方式安装，它能更好地和 [webpack](https://webpack.js.org/) 打包工具配合使用。
```
npm i zch-gulu -S
// or yarn
yarn add zch-gulu
```
